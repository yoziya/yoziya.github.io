RectTransform

不属于 UGUI 而属于 UnityEngine，继承自 Transform。

Anchor

绝对布局

当anchorMax与anchorMin相等时，Anchor呈现为一个锚点，此时为绝对布局。

  • PosX、PosY、PosZ为Pivot到锚点的距离

  • Width、Height为UI的恒定大小

相对布局

当anchorMax与anchorMin不相等时，Anchor呈现为一个锚框，此时为相对布局。

  • PosZ为到父物体在Z轴的偏移
  • Left、Right、Top、Bottom为相对于父物体锚框的边的距离

Pivot

UI元素旋转缩放的中心点

Offset

sizeDelta

OffsetMax - OffsetMin的值

锚点情况下可以设置UI大小

Canvas

可视化

Image和RawImage

RawImage支持UV操作，支持除Sprite之外更多类型的贴图，少了很多复杂功能，适合仅展示图片

Text

交互

实现 Selectable 类和继承了 IEventSystemHandler 的互动事件接口

EventSystem

UGUI基础优化

减少 DrawCall

DrawCall是 CPU 计算好顶点坐标、UV、颜色等并通知显卡绘制的过程。DC 大于 CPU 与 GPU的通讯带宽时会影响性能，移动端开发一般控制在 150 以内。
